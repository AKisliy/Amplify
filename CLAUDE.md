# Amplify VKR — Claude Instructions

## Wiki

The project has a wiki (Obsidian vault) at `wiki/`. It is the primary knowledge base for architecture, issue plans, research, and implementation decisions.

At the start of every session, read:
1. `wiki/CLAUDE.md` — schema and operating rules for the wiki
2. `wiki/issues/index.md` — one row per issue, current statuses
3. `wiki/issues/AMPLIFY-N/plan.md` and `research.md` for the active issue

See `wiki/CLAUDE.md` for the full directory structure, file templates, and maintenance rules.
