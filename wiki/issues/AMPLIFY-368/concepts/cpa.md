# Cost Per Asset (CPA)

> Zone 1 — Input Matrix | Ultimate Unit Economic

**Cost Per Unique Asset (CPA - Generation)** — the master KPI for Zone 1, representing the ultimate unit economic cost of generating a single verified video. It normalizes varying API costs, failed generations, and varying template lengths into a single number.

`CPA = Total Content Spend ÷ Output Volume`

## The Human Benchmark and Arbitrage

CPA proves the core value proposition of the Amplify system (automation vs. human labor).

- **Human Baseline:** The estimated cost to produce one video manually (e.g., $15.00).
- **AI CPA:** The actual cost generated by the system (e.g., $1.50).
- **Arbitrage Margin (%):** The percentage saved per video. Example: `$13.50 saved / $15.00 baseline = 90% Arbitrage Margin`.
- **Accumulated Savings (Total Capital Saved):** The absolute dollar amount saved over a period (e.g., $13.50 saved × 1,000 videos = $13,500).

## Diagnostic Value

- **Trailing Average vs Daily Volatility:** Daily CPA fluctuates due to API spikes or failure rates. A trailing average (e.g., 7-day or 30-day) provides a stable, representative metric.
- **Margin Visualization:** Tracking the gap between the Human Baseline and AI CPA visualizes the absolute margin of leverage for stakeholders.
